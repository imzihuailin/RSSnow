# RSSnow

一个**自己说了算**的 RSS 阅读器。很多博客不能调字体、不能调背景，读起来费劲；RSSnow 让你想怎么舒服就怎么读。

## 为什么用 RSSnow？

- **字体随你选**：宋体、黑体、楷体、微软雅黑、Georgia、Arial……正文和标题都能调
- **背景随你选**：纯白、纯黑、米黄、护眼绿，保护眼睛
- **没有正文也能读**：RSS 只给标题？自动帮你抓原文，照样在阅读器里看
- **读到哪记到哪**：进度条随时拖动，下次打开自动回到上次位置

## 怎么用？

1. 在项目目录运行 `npm install` 和 `npm run dev`，用浏览器打开
2. 点「添加 RSS」，把博客的订阅地址粘进去
3. 点订阅卡片进文章列表，点文章开始读
4. 在读页里点一下，底部会出现进度、字体、背景设置，按自己习惯调

## 适合谁？

- 经常看博客、新闻订阅的人
- 觉得原站字体太小、太刺眼的人
- 想在一个地方集中读很多 RSS 的人

数据都存在浏览器本地，不上传、不联网同步，完全在你电脑里。
